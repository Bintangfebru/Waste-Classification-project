## Waste Classification System using CNN
#### Overview
This project presents the development of an image classification system using Convolutional Neural Networks (CNN) to categorize waste into three classes: Anorganik, Organik, and Toxic. The model leverages transfer learning to achieve reliable performance and demonstrates the practical application of deep learning in environmental sustainability
The system is designed to support automated waste sorting.

#### Key Features
- Image classification using CNN with transfer learning
- Data augmentation to improve generalization
- Model evaluation using accuracy, precision, recall, and F1-score
- Confusion matrix visualization for performance analysis
- Modular and scalable project structure

#### Methodology
###### Dataset
The dataset used in this project was obtained from the public dataset available on Kaggle
 and consists of waste images categorized into three classes:

- Anorganik 3410
- Organik 2539
- Toxic 2158

The dataset was divided into training, validation, and testing subsets to ensure proper model evaluation and generalization performance.

Dataset Distribution
Subset	Total Images
- Training 6482

###### Data Preprocessing
- Image resizing to uniform dimensions
- Pixel normalization
- Data augmentation (rotation, zoom, flip, brightness adjustment)
###### Model Development
- Transfer learning using MobileNetV2
- Fully connected layers for classification
- Dropout for regularization
###### Model Evaluation
- Accuracy
- Precision
- Recall
- F1-score
- Confusion matrix

#### Result
###### classification-report
![result](img\classification-report.png)
The model demonstrates strong overall performance, achieving an accuracy of 83% across 811 samples. However, its effectiveness varies significantly across waste categories.

- Anorganik shows excellent predictive power with high precision (0.94) and recall (0.99), indicating that the model reliably identifies this class with very few false negatives.

- Organik has moderate performance. While precision is acceptable (0.84), recall is notably low (0.61), suggesting the model misses nearly 40% of actual Organik samples, likely misclassifying them as other categories.

- Toxic exhibits the lowest precision (0.66) despite reasonable recall (0.82), meaning the model frequently produces false positives for this critical class, incorrectly labeling non-Toxic waste as Toxic.

The macro and weighted averages (F1-scores of 0.80 and 0.82, respectively) confirm balanced overall performance, but the low recall for Organik and low precision for Toxic highlight specific weaknesses. Recommendations include collecting more Organik samples and reviewing features causing Toxic misclassifications to improve precision and reduce false alerts.

###### confusion matrix
![matrix](img\confusion-matrix.png)
confusion matrix reveals that the model performs excellently for the Anorganik class with almost no errors, but has two critical weaknesses that require my immediate attention. The most significant issue is that 90 actual Organik samples are misclassified as Toxic, which drastically lowers recall for Organik and artificially inflates false positives for Toxic – this could lead to unnecessary toxic waste handling costs and reduced trust in the system. Additionally, 28 Toxic samples are misclassified as Organik, further compromising precision for the Toxic category. While the Anorganik class needs no improvement, I must prioritize reducing the Organik-to-Toxic confusion by analyzing feature overlaps and adjusting decision boundaries. Until these issues are resolved, I consider the model reliable only for Anorganik detection, and not yet fully dependable for separating Organik from Toxic
